# Session Summary Template

## Purpose

Use this template to summarize an OpenClaw work session, capture outcomes, and make handoff or continuation easier.

## Recommended placeholders

- `{{context}}`: Background for the session or project
- `{{task}}`: Main work performed during the session
- `{{goal}}`: Intended outcome of the session
- `{{work_completed}}`: Key actions, edits, or decisions made
- `{{artifacts}}`: Commits, files, logs, screenshots, or links
- `{{open_items}}`: Remaining issues, blockers, or follow-ups

## Template

```md
You are summarizing an OpenClaw work session.

Context:
{{context}}

Task:
{{task}}

Goal:
{{goal}}

Work completed:
{{work_completed}}

Artifacts:
{{artifacts}}

Open items:
{{open_items}}

Instructions:
1. Summarize the session in clear, factual language.
2. Highlight outcomes, not just activity.
3. Capture important decisions and why they were made.
4. Note any unresolved risks, blockers, or dependencies.
5. End with concise next steps so another agent can continue smoothly.

Expected output format:
- Session overview:
- Goals achieved:
  - ...
- Work completed:
  - ...
- Key decisions:
  - Decision:
    - Reason:
- Artifacts produced:
  - ...
- Open issues:
  - ...
- Recommended next steps:
  1. ...
  2. ...
- Handoff notes:
```
