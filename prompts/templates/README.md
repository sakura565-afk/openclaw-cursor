# Prompt templates

Reusable, structured prompts for coding workflows and self-improvement. Each file documents placeholders, when to use the template, and a filled example.

## Convention

- **Placeholders** use `{{LIKE_THIS}}`. Replace them with your context before sending the prompt.
- **Primary sections** in each template: purpose, placeholders table, usage example, then the prompt body.
